Shows random Bible Verses as widget or using shortcode on each page visits.

== Description ==

This plugin will create a widget that will show bible verses in random order. Each time user visits it will show different verses from dailyverses.net.

Apart from widget you can use shortcode [js-bible-verses]. It can be used anywhere in the page. It will also show verses.

Visit <a href="http://www.joelsays.com/">Plugin Official Website</a>


Features of Bible Verses are,

 * You can use SHORTCODE to show random bible verses.

== Installation ==

1. Upload `Bible Verses` plug-in zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will see a new Widgets on your widgets manager.
4. Drag and Drop widget to the position where you want to show bible verses.
5. If you want to show bible verses without widgets in pages or posts, use shortcode [js-bible-verses].
